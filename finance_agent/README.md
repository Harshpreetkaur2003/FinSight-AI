# 💰 FinSight AI — Expense Intelligence & Financial Health Agent

A production-ready AI agent that analyzes your transaction data, categorizes expenses using ML,
detects overspending, identifies anomalies, and acts as a conversational financial advisor.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the ML model
```bash
python train_model.py
```

### 3. Launch the app
```bash
streamlit run app.py
```

### 4. Open in browser
Navigate to `http://localhost:8501` and either:
- Click **"Load Sample Data"** in the sidebar for an instant demo, or
- Upload your own CSV file

## 🧠 Features

### ML Expense Categorization
- TF-IDF (bigrams) + RandomForest pipeline
- Categories: **Food, Rent, Transport, Shopping, Bills, Others**
- Falls back to keyword rules when model is not found

### Analysis Engine (`agent/analyzer.py`)
- Monthly totals and averages
- Category-wise breakdown
- Linear trend detection across months
- Savings rate estimation
- Per-category budget comparison

### Anomaly Detection
- Z-score based transaction flagging (threshold: 2.5σ)
- Spending spike detection by day
- Visual scatter plot with anomalies highlighted

### Risk Scoring (0–100)
| Score | Level  | Meaning                               |
|-------|--------|---------------------------------------|
| 0–39  | Low    | Finances healthy                      |
| 40–64 | Medium | Some areas need attention             |
| 65–100| High   | Immediate corrective action needed    |

Risk factors: savings rate, overspending categories, trend direction, anomaly frequency.

### Conversational AI Advisor
- Rule-based engine works out of the box — no API key needed
- Add OpenAI API key in sidebar to upgrade to GPT-3.5-turbo via LangChain
- Supports natural language questions:
  - *"Where am I overspending?"*
  - *"How can I save money?"*
  - *"Analyze my expenses"*
  - *"What is my risk score?"*

### Export Options
- 📄 **CSV** — Categorized transaction data
- 📊 **Excel** — Multi-sheet: transactions, monthly summary, category totals, recommendations, anomalies
- 📋 **Text Report** — Human-readable financial health report

## 🔧 Extending

**Add a new category:**
1. Add labeled rows to `data/training_data.csv`
2. Add keywords to `KEYWORD_RULES` in `models/predict.py` and `train_model.py`
3. Add budget % to `CATEGORY_BUDGETS` in `agent/analyzer.py`
4. Add tips to `SAVING_TIPS` in `agent/rules.py`
5. Re-run `python train_model.py`

**Use a different LLM:**
Edit `agent/llm_agent.py` — replace `ChatOpenAI` with any LangChain-compatible model
(e.g., `ChatAnthropic`, `ChatGoogleGenerativeAI`, `ChatOllama`).
